### Instruction:

1) git clone https://github.com/NikSko0r/python-project-49.git
2) poetry install
3) poetry build
4) python3 -m pip install --user dist/*.whl

### Commands to run from the console:

1) brain-progression
2) brain-gcd
3) brain-calc
4) brain-even
5) brain-prime
